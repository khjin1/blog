MyBlog : AI 기반 블로그 플랫폼
개인 프로젝트 : 김혜진

1. 프로젝트 소개
서비스명: MyBlog

서비스 한줄 소개: AI 기술을 접목하여 누구나 쉽게 블로그 포스팅을 시작할 수 있도록 돕는 정적 블로그 플랫폼입니다.

2. 기획 배경
기술적 도전: 라이브러리와 API 연동을 통해 프론트엔드 환경에서 블로그 플랫폼의 핵심 기능을 직접 구현해 보고자 기획되었습니다.

목표: 사용자 인증부터 AI 콘텐츠 생성까지, 실제 서비스에서 필요한 흐름을 경험하는 것에 중점을 두었습니다.

3. 주요 기능 소개
AI 포스팅 초안 생성
사용자가 제목을 입력하면 OpenAI를 통해 관련 있는 블로그 본문 초안을 자동으로 작성합니다.

사용자 인증 및 관리
회원가입 및 로그인: 프론트엔드 로직 기반의 사용자 인증 시스템을 구현했습니다.

마이페이지: 로그인 유무에 따라 접근 권한을 제어합니다.

콘텐츠 브라우징 및 작성
홈페이지: 전체 블로그 포스트 목록을 한눈에 확인할 수 있습니다.

상세 페이지: 개별 포스트의 내용을 읽을 수 있는 상세 뷰를 제공합니다.

작성 페이지: 감각적인 UI 환경에서 글을 작성하고 AI의 도움을 받을 수 있습니다.

4. 트러블슈팅 경험
"혼자서 해결해 나가는 힘"
모든 것을 혼자 기획하고 개발하는 과정에서 사소한 문법 에러부터 배포 환경 설정까지 많은 어려움이 있었습니다. 하지만 공식 문서 확인, 커뮤니티 활용, 그리고 논리적인 디버깅을 통해 마주한 문제들을 하나씩 천천히 해결하며, 스스로 에러를 해결하는 프론트엔드 개발자로서의 기초 체력을 길렀습니다.

5. 앞으로 더 하고 싶은 것들
현재는 개인 프로젝트로 진행했지만, 기회가 된다면 팀원들과 협업하여 Git 워크플로우를 경험하고 더 복잡한 기능을 함께 설계해 보고 싶습니다.
